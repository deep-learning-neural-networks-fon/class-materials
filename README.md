Ovaj repozitorijum sadrži materijale sa predavanja i vežbi za predmet Duboko učenje i neuronske mreže.

Predmet se izvodi kao izborni na Fakultetu organizacionih nauka, Univerzitet u Beogradu, na četvrtoj godini studijskog programa Informacioni sistemi i tehnologije (modul: Softversko inženjerstvo).
